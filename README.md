# endchess-contracts

Shared TypeScript contracts for EndChess:

- `settings/` — puzzle and mistakes DTOs (formerly `endchess-api-settings`)
- `queue/` — RabbitMQ queue names and message shapes

Previously published as `endchess-api-settings@1.x`.

Developed in the endchess2 workspace with sibling `file:../` paths on the `dev2` branch.
